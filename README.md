# 민재의 호기심방 · Wonder Room

세상의 원리를 시뮬레이션과 인터랙티브 비주얼로 옮겨둔 아카이브.

## 새 작품 추가하는 방법

1. `works/` 폴더에 새 HTML 파일을 업로드
   - 파일명 규칙: `YYYY-MM-DD_작품이름.html`
   - 예: `2026-05-01_black-hole.html`

2. `works.json` 파일을 열고, `works` 배열의 맨 앞에 새 항목 추가:

```json
{
  "id": "black-hole",
  "title": "블랙홀 중력 렌즈 시뮬레이션",
  "description": "한 줄 설명",
  "subjects": ["천문학", "물리학"],
  "tags": ["3D", "인터랙티브"],
  "file": "works/2026-05-01_black-hole.html",
  "date": "2026-05-01",
  "emoji": "🕳️",
  "color": "#2a1a3a",
  "note": ""
}
```

3. 저장하면 Vercel이 1~2분 내 자동 배포합니다.

## 필드 설명

| 필드 | 설명 | 예시 |
|---|---|---|
| `id` | 내부 식별자 (영문) | `"solar-3d"` |
| `title` | 카드 제목 | `"태양-지구 3D 공전 시뮬레이션"` |
| `description` | 2~3줄 설명 | `"지구가 자전축을..."` |
| `subjects` | **주제 태그 (복수)** | `["천문학", "지구과학"]` |
| `tags` | 형식 태그 | `["3D", "인터랙티브"]` |
| `file` | HTML 파일 경로 | `"works/...html"` |
| `date` | 작성일 (YYYY-MM-DD) | `"2026-04-18"` |
| `emoji` | 썸네일 이모지 | `"🌍"` |
| `color` | 썸네일 배경색 (hex) | `"#1a3a5a"` |
| `note` | 개인 메모 (선택) | `"만든 계기..."` |

## 주제 예시
천문학 / 물리학 / 생물학 / 지구과학 / 화학 / 수학 / 기하 / 역사 / 지리 / 예술 / 음악 / 건축 / 언어 / 문학 / 기술 / 공학

필요할 때 새 주제는 자유롭게 추가 — `subjects`에 쓰기만 하면 자동으로 필터 칩으로 나타납니다.
